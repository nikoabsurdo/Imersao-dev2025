Destino Certo — Interface Web para Exploração de Destinos Turísticos

O Destino Certo é uma aplicação web desenvolvida com foco em simplicidade, responsividade e boa experiência do usuário. O projeto apresenta uma interface moderna para consulta e visualização de destinos turísticos, priorizando organização visual, navegação intuitiva e acessibilidade.
A estrutura do site foi construída utilizando HTML, CSS e boas práticas de design, garantindo compatibilidade com diferentes dispositivos e uma apresentação clara das informações.
